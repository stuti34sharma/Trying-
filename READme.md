# This is a project to test your Reaction time. 

A simple Python game to test your reaction speed.  
Press **Enter** as quickly as possible when prompted and see how fast your reactions are!  

---

## Features
- Random delay before each round.
- Records reaction time for **5 rounds**.
- Shows **quickest**, **slowest**, and **average** reaction times.
- Interactive and fun to play!

---
